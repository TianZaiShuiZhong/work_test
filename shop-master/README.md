# shop

#### 介绍
[铁职院]创新Web商城项目 是为广州铁路职业技术学院编写的实践类应用。

任何问题可发邮件沟通：pangqr@foxmail.com

#### 软件架构

1. SpringBoot 3
2. MySQL 8


#### 安装教程

1.  将SQL数据库文件导入本地MySQL
2.  将本项目下载到本地通过IDE打开，建议IDE：IntelliJ IDEA/ Eclipse / VSCode ...


#### 使用说明

1.  修改项目中application.yml文件中数据库连接信息
2.  运行项目
